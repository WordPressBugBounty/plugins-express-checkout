== Description ==

**PayPal for WooCommerce** is the official solution for integrating PayPal Express Checkout and additional payment options into your WooCommerce store. Developed by an official PayPal Partner, this plugin ensures you can offer PayPal, credit cards, and local payment methods, providing a secure and reliable payment experience for your customers.

With **PayPal for WooCommerce**, you can enable:
* PayPal Smart Payment Buttons
* Pay in 4 (Buy Now, Pay Later)
* Credit and Debit Card Payments with 3D Secure
* Venmo, iDEAL, Sofort, and other local payment methods

=== Key Features of PayPal for WooCommerce ===

== Installation ==

= Automatic Installation =

1. Log in to your WordPress dashboard.  
2. Navigate to **Plugins > Add New**.  
3. Search for "Express Checkout via PayPal for WooCommerce".  
4. Click **Install Now**, then **Activate**.  
5. Configure your PayPal settings under **WooCommerce > Settings > Payments**.

= Manual Installation =

1. Download the plugin’s zip file.  
2. Extract the zip file and upload the plugin folder to `/wp-content/plugins/`.  
3. Go to **Plugins** in your WordPress dashboard and activate the plugin.  
4. Set up PayPal credentials in **WooCommerce > Settings > Payments**.

== Frequently Asked Questions ==

= How do I configure PayPal for WooCommerce? =  
After activating the plugin, go to **WooCommerce > Settings > Payments**, select **PayPal for WooCommerce**, and enter your PayPal API credentials (Client ID and Secret). Save your changes, and you're ready to go.

= How can I test payments using sandbox accounts? =  
To test transactions:
1. Log in to [PayPal Developer](https://developer.paypal.com) and create sandbox accounts.
2. Use sandbox API credentials in the PayPal settings under **WooCommerce > Payments**.
3. Place test orders with sandbox buyer accounts.

= Does PayPal for WooCommerce support subscriptions? =  
Yes! The plugin integrates with WooCommerce Subscriptions, allowing you to offer recurring payment options for your customers.

= What payment methods are supported by PayPal for WooCommerce? =  
Along with PayPal, this plugin supports major credit cards like Visa, Mastercard, American Express, and local payment methods such as Venmo, Sofort, Bancontact, giropay, and iDEAL.
